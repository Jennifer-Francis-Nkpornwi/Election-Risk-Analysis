# Election Intergrity Dashboard: Predictive Risk Analysis

## Overview

This project analyzes polling unit election data and election observer incident reports from the 2019 and 2023 election cycles to identify LGAs that may be at higher risk of electoral irregularities.

The project combines data cleaning, data integration, anomaly detection, risk modeling, and dashboard development to support data-driven election monitoring.

## Project Objective

* Merge polling unit and incident report datasets.
* Identify voting anomalies and incident patterns.
* Develop a Risk Score model for each LGA.
* Compare electoral risk between 2019 and 2023.
* Build an interactive Power BI dashboard for monitoring high-risk areas.

## Tools Used

* Excel Power Query
* Power BI
* DAX
* Excel

## Key Findings

* Highest-Risk LGAs: Nassarawa, Port-Harcourt, and Surulere.
* Most Improved LGAs: Port-Harcourt, Surulere, and Nassarawa.
* A relationship was observed between voting anomalies and reported election incidents across several LGAs.
